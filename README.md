# Дискорд бот: очередь
## Setup:
1) Создайте папку "configs"
2) В ней, файл botConfig.json
3) Заполните его следующим образом

      `     {
          "token": "(1)",
          "botId": "(2)",
          "csgoQueueChannel": "(3)"
      }`
    
   (1) - токен бота, 
   
   (2) - id бота, 
   
   (3) - id текстового канала (канал должен быть пустым). 
   
   (2) и (3) можно получить в самом дискорде в режиме разработчика.
   
 4) `npm install && node .`
